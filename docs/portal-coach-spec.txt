北陽男子ミニバス ポータルコーチ仕様メモ
作成日: 2026-07-08

概要
ポータルコーチは、北陽男子ミニバスの保護者がバスケットボールやミニバスの基本を確認できる学習ページです。
初期版ではAIを使わず、悩みカテゴリを選んでから具体的な問いへ進む「選択式チャット風UI」と、ルール確認クイズを掲載します。
保護者の経験差を埋め、試合観戦・当番・TO・子どもへの声かけの不安を減らすことを目的にします。

初期版で扱う内容
1. ポータルコーチ本体
保護者が悩みを選び、次に具体的な問いを選ぶ。回答は短く表示する。

悩みカテゴリ
ルールがわからない
用語がわからない
試合の見方を知りたい
TOが不安
声かけに迷う
北陽の運用を確認したい

2. ルール確認クイズ
保護者が短時間で確認できる選択式。50問のCSVからランダムに2問表示する。
「別の問題」ボタンで表示問題を切り替えられる。
クイズはGoogleスプレッドシートから読み込む。
スプレッドが一時的に読めない場合は、予備としてサイト内CSVを読み込む。
保護者向け画面には管理用スプレッドリンクを出さない。編集は管理者・担当者がDrive内のスプレッドで行う。

クイズCSVの列
id: 管理番号
difficulty: 難易度。やさしい、ふつう、高め。任意。列がない場合は画面に表示しない。
category: 分類
question: 問題文
choice_a: 選択肢A
choice_b: 選択肢B
choice_c: 選択肢C
correct: 正解の選択肢。A、B、Cのいずれか。
explanation: 回答後に表示する短い説明
source_folder: 根拠にする資料フォルダ

ヒアリング後に北陽の運用へ合わせて更新する。

今後追加する内容
1. よくある質問
集合、欠席連絡、当番、配車、持ち物、写真動画共有など。北陽チーム資料を準備してから反映する。

2. ミニバス初めての保護者向け
試合当日の流れ、保護者の役割、持ち物、応援マナー、連絡先、見るべき資料を整理する。

3. 北陽の試合の見方
コーチが大切にしている見方、子どもへの声かけ、成長の見つけ方をチーム方針としてまとめる。

4. 資料根拠の明記
各説明に対して、どの資料フォルダ・どのPDFを根拠にしたかを残す。

運用ルール
公式ルールと北陽独自運用は混ぜず、ページ内でも区別する。
細かい判定や大会ごとの運用は、公式資料・大会要項・審判判断を優先する。
北陽チーム資料が未整備の項目は、断定せず「準備中」または一般的な説明に留める。

AI化する場合の考え方
将来的にAIチャット化する場合は、Drive内資料を根拠にした回答に限定する。
資料にない内容は「資料では確認できません」と返す設計にする。
GitHub PagesだけではAPIキーを安全に扱えないため、AI化には別途バックエンドが必要。
